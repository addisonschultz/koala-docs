## Installation

### Step 1 
Find your Public API Key here: https://app.getkoala.com/goto/settings/install

<Frame>
![](/images/integrations/your-public-key.png)
</Frame>

### Step 2
Add the following to your `mint.json` file:

```json
{
  "navigation": [...],
  "logo": [...],
  [...],
  [...],
  [...],
  "analytics": {
    "koala": {
      "publicApiKey": "pk_YOURKEYHERE"
    }
  }
}
```
